# Countdown Timer

A simple countdown timer web app built using: 
- FastAPI for the backend
- HTML and JavaScript for the frontend

Users input a time, start a countdown, and receive an alert sound when the timer reaches zero.

It also has a feature to save previously used times.

## Setup

```bash
pip install -r requirements.txt
```

## Run

```bash
python -m uvicorn main:app --reload
```

Opens at [http://localhost:8000](http://localhost:8000)
